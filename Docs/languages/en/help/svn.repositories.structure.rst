Structure in the Repositories
=============================

It is often desirable to use a purely *PHP*-based configuration file. The following code illustrates how easily this
can be accomplished:

.. code-block:: text

    Repositories/
        /Example
            /logs
            /Post
            /Pre
            /Start
        /Example2
            /logs
            /Post
            /Pre
            /Start

.. _`subversion`: http://subversion.apache.org/
